Personal Budgetplanner Excel

About The Planner

This is an interactive budget planner that is semi-automated. You put in your salary, bills, expenses and deposit you want to make for dept/savings each month. All visuals and calculations is automated and changes over time as you add new data to the spreadsheets.


I created this planner to be visually appealing and at the same time easy to use with the interactive menu that lets you navigate without the tabsbar.
